# MCC Core

MCC Core is the central intelligence of Minecraft Control Center.

It communicates with the Fabric Bridge, processes Minecraft data, manages commands, and runs automation logic.

## Responsibilities

- Manage WebSocket connections
- Receive telemetry and events
- Send commands to the Fabric Bridge
- Validate responses
- Manage profiles
- Store data with SQLite
- Run automation workflows
- Prepare Lua scripting support
- Prepare plugin support
- Communicate with the web dashboard
- Manage logs and configuration

MCC Core will be developed in Python.
