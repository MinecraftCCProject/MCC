# MCC Roadmap

Minecraft Control Center is currently in early development.

## Phase 1 — Fabric Bridge

- Fabric mod for Minecraft Java 1.21.11
- Local WebSocket communication
- Versioned communication protocol
- Player telemetry
- Inventory data
- Block scanning
- Entity scanning
- Screenshot capture
- Movement commands
- Interaction commands
- Emergency stop
- Command validation
- Capability reporting
- Logging and configuration

## Phase 2 — MCC Core

- Python application core
- Command and event management
- Minecraft connection management
- Automatic reconnection
- Configuration profiles
- SQLite database
- Internal API
- Automation engine
- Plugin-ready architecture
- Lua scripting preparation

## Phase 3 — Web Dashboard

- Main dashboard
- Matrix-style console
- Inventory viewer
- Manual controls
- Statistics
- Profiles
- Settings
- Responsive interface
- Multiple themes
- French and English support

## Phase 4 — Advanced Features

- Mini-map
- Waypoints
- Path editor
- Blockage detection
- Lua scripting engine
- Discord webhook integration
- Advanced statistics
- Data export
- Plugin system

## Phase 5 — MCC Ecosystem

- Secure remote relay
- VPS and Raspberry Pi support
- Remote dashboard access
- Multiple Minecraft instances
- Public documentation
- Official bilingual blog
- Official Discord community

## Phase 6 — MCC Launcher

- MCC installation
- Fabric installation
- Mod installation
- Component updates
- Profile management
- Minecraft launch
- MCC Core launch
- Dashboard launch

## Future Vision

- Multiple Minecraft versions
- Community scripts
- Community plugins
- Intelligent automation
- Optional mobile interface
